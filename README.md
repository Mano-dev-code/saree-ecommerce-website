# 🛍️ Meena Sarees – E-commerce Website

A responsive front-end e-commerce website designed to showcase different saree collections with a clean product layout and modern user interface.

## 📌 Overview

Meena Sarees is a front-end web application that allows users to explore various saree collections through an intuitive and visually appealing interface. The project focuses on delivering a smooth browsing experience with organized product sections and responsive design.

## 🚀 Features

- Product showcase with saree images and details  
- Category-based browsing and filtering functionality  
- Clean and structured product card layout  
- Responsive design for mobile, tablet, and desktop  
- Interactive UI elements such as hover effects and animations  

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Responsive Design (Flexbox & Grid)
