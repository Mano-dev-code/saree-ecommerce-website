# Meena Sarees – E-commerce Website

Meena Sarees is a front-end e-commerce website designed to showcase and sell different varieties of sarees through a clean and user-friendly interface. The website allows users to explore collections, view product details, and browse sarees based on categories.

The goal of this project is to create an attractive and responsive online shopping experience for customers while practicing modern web development concepts.

## Features

* Product showcase with saree images and details
* Category and filter options for easy browsing
* Clean product card layout
* Responsive design for different screen sizes
* Interactive UI elements such as hover effects and animations

## Technologies Used

* HTML
* CSS
* JavaScript
* Basic responsive design techniques

## Purpose of the Project

This project was created as a portfolio project to practice front-end development and build a real-world style e-commerce interface.
